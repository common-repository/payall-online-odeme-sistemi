== Description ==

PayALL WooCommerce modülünü web sitenize hızlı ,kolay bir şekilde entegre edip  dakikalar içerisinde ödeme almaya başlayabilirsiniz.

[www.payall.com.tr](https://www.payall.com.tr/)

= PayALL’un Avantajlı Dünyasında Neler Var? =

1. PayALL Sanal POS üzerinden yaptığınız tahsilatlar ertesi gün hesabınıza yansır.
1. Yurt dışından güvenli ödeme almak için de PayALL’u kullanabilirsiniz.
1. Herhangi bir kurulum gerektirmeyen link ile ödeme yöntemini anında kullanmaya başlayabilirsiniz.
1. Türkiye’deki bankalara ait kredi kartları ile taksitli ödeme alma imkanı.
1. Kurulum ücreti , aidat  ve  gizli ücretler yoktur.

= Gereksinimler =
* Payall WooCommerce eklentisiyle ödeme alabilmeniz çin PayALL’ da bir üye iş yerinizin olması gerekmektedir.


Requirements
PHP 5.4 and greater.
cURL
WooCommerce 3.5 requires WordPress 3.5+


== Installation ==

Aşağıdaki adımları takip ederek kurulumu gerçekleştirebilirsiniz.

1.	Payall.com.tr >Üye İşyeri >Entegrasyon sekmesi altında PayALL WooCommerce eklentisini indiriniz.
1.	İndirdiğiniz .zip dosyasını Wordpress admin panel >Eklentiler >Yeni ekle > Eklenti Yükle alanına yükleyiniz.
1.	Client id ve client secret bilgilerini girmek için WooCommerce >Ayarlar >Ödemeler sekmesi altında PayALL Online Ödeme Sistemi yönet butonuna tıklayarak size verilen client id ve client secret bilgilerini girip üye iş yerinize göre kişileştirin.
1.	PayALL sanal pos kullanıma hazır.
